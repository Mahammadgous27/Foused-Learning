## 📌 Smart Screen Time Tracker App

### 1. Overview
The **Focused Learning Application** is designed to help users improve focus by monitoring app usage and reducing distractions. It tracks screen time, classifies apps, and provides timely notifications to control excessive usage.

---

### 2. Features
- Tracks usage time of applications  
- Classifies apps as **Working** or **Distracting**  
- Continuously monitors app usage  
- Sends alerts when usage exceeds limits  
- Displays daily/weekly usage summary  

---

### 3. Working Process

1. **App Launch & Permission**  
   Requests usage access permission from the user.

2. **Fetch App Usage**  
   Uses `UsageStatsManager` to track time spent on each app.

3. **Classify Apps**  
   Identifies apps as *Working* or *Distracting*.

4. **Monitor Usage**  
   Continuously tracks app usage duration.

5. **Check Threshold**  
   Compares working app usage with predefined limits.

6. **Send Intervention**  
   Triggers a notification if usage exceeds the threshold.

7. **Display Summary**  
   Shows daily or weekly usage statistics.

---

### 4. Objective
To promote focused learning by reducing distractions and improving time management.
